RTL8722DM_mini 快速入門¶

環境配置¶

AmebaD RTL8722DM_MINI 目前支持32位和64位 Windows XP/7/8/10，Linux 和 Mac 操作系統。 在本文檔中，請使用使用 1.8.12 或更高版本的 Arduino IDE。

AmebaD RTL8722DM_MINI 簡介¶

Ameba 是一個用於開發所有類型的物聯網應用，易於編程的平台。 AmebaD 配備了包括 WiFi、GPIO INT、 I2C、UART、SPI、PWM、ADC 各種外圍接口。通過這些接口，AmebaD可以與如 LED、開關、壓力計、濕度計 PM2.5粉塵傳感器等電子元件連接。採集的數據可以通過 WiFi上傳，並被智能設備上的應用程序利用， 以實現物聯網的實施。
